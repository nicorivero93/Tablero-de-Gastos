# **App Name**: BudgetWise

## Core Features:

- Credit Card Management: Create and manage credit cards, including setting credit limits and tracking balances.
- Expense Categorization: Categorize expenses into predefined or custom categories for detailed tracking.
- Tagging System: Tag expenses with custom labels for more granular analysis and filtering.
- Financial Dashboard: Visualize income, expenses, and savings in a clear pie chart.

## Style Guidelines:

- Primary color: Soft blue (#A0CFEC) to convey trust and stability.
- Background color: Light, desaturated blue (#F0F8FF).
- Accent color: Muted green (#8FBC8F) for income highlights and positive savings indicators.
- Body and headline font: 'PT Sans' for a modern, approachable feel.
- Use clear, minimalist icons for categories and tags.
- Subtle transitions and animations when updating the dashboard or adding expenses.
